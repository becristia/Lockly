# Secure Box 项目评估报告

**日期：** 2026-05-17

## 执行摘要

Secure Box 不再是玩具密码笔记本。安全架构比典型 MVP 更强：Argon2id、AES-256-GCM、KEK/DEK 分层、每条记录随机数、生物识别回退语义、清单完整性、回滚锚点、自动锁定、剪贴板清理和主密码策略都已具备。

主要弱点是产品成熟度。该项目目前感觉像一个安全的工程样本，而不是日常使用的密码管理器。加密路径是严肃的，但面向用户的安全态势未被充分解释，UI 功能齐全但朴素，密码健康状况、生成器预设和备份可用性等实际工作流程仍然薄弱。

## 评分

- 逻辑完整性：8/10
- 安全态势：8.5/10
- 视觉打磨：6/10
- 实际可用性：6.5/10
- MVP 完整性：8/10
- 长期产品潜力：7/10

## 逻辑审查

应用程序具有连贯的本地优先流程：

- 首次运行设置主密码。
- 主密码解锁和生物识别快速解锁。
- 加密保险库存项的 CRUD。
- 密码生成器到保存流程。
- 生物识别、自动锁定、剪贴板清理、备份导入/导出、清除本地保险库的设置。
- 围绕关键变异的清单和回滚锚点检查。

代码库也合理地分解了：

- `lib/core/crypto/` 拥有加密和 KDF 行为。
- `lib/core/vault/` 拥有保险库语义和完整性检查。
- `lib/core/backup/` 拥有备份导入/导出。
- `lib/core/security/` 拥有生命周期、自动锁定和主密码策略。
- `lib/features/*` 拥有屏幕。

尖锐批评：

- 搜索解密每个活动项并在内存中过滤。这遵循了"无明文索引"约束，但随着项目数量增长会退化。未来需要安全的内存索引或解锁会话缓存。
- 面向用户的错误类别太粗糙。错误的主密码、损坏的数据、回滚检测、备份格式错误和生物识别失败不应该都感觉像"稍后再试"。
- 功能流程完整，但未优化。用户可以完成任务，但界面尚未减少常见操作的重复摩擦。

## UI 和视觉审查

UI 干净，对于安全工具来说适当克制。蓝色/绿色信任调色板是合理的，项目避免了会削弱感知安全的活泼或装饰性图案。

尖锐批评：

- 该应用仍然接近默认 Flutter Material。卡片、设置行、详情行和表单页面功能齐全但无特色。
- 没有深色模式，这使得密码管理器的应用感觉不够成熟。
- 安全状态不够可见。用户应该看到保险库何时锁定、备份何时验证、密码何时弱/重用以及剪贴板清理何时待处理。
- `SecureScaffold` 用于设置/生成器/设置，而列表/详情/编辑使用原始 `Scaffold`。这造成了微妙的视觉不一致。
- 生成器屏幕是一个表单，还不是一个工具。缺少强度解释、预设、以复制为优先的行为，以及为生成的密钥更强的视觉层次。

## 安全审查

强有力的决定：

- 主密码未存储。
- KEK/DEK 分层是正确的。
- 主密码轮换重新包装 DEK，而不是重新加密所有项目。
- 新保险库和轮换保险库使用 Argon2id。
- 使用生成的随机数使用 AES-256-GCM。
- 生物识别解锁不替换主密码。
- 应用在超时/后台时清除待处理密码剪贴板数据。
- 清单和回滚锚点检查防御有意义的本地回滚类。
- 主密码强度现在拒绝常见的弱模式。

残余风险：

- 解密的条目字段成为 Dart `String` 值和 `TextEditingController` 文本。这些无法可靠地清零。应用应最小化它们保持引用的时间。
- 备份导出 UX 是原始的。向用户显示 JSON 适用于 MVP，但鼓励不安全的复制/粘贴习惯。
- 应用没有密码健康检查，因此无法警告重复、弱、旧或网站相似的密码。
- 解锁节流存在，但围绕锁定/冷却的用户体验可能更清晰。
- 完整性失败需要不同的、高严重性的 UI。静默通用失败对于安全产品来说是不够的。

## 实际可用性审查

该应用适用于小型个人保险库。随着记录数量增长，它变得不那么符合人体工程学。

缺失或开发不足的实际功能：

- 本地密码健康报告。
- 重复密码检测。
- 密码年龄跟踪和陈旧密码警告。
- 生成器预设，用于默认强密码、可读密码短语和网站兼容密码。
- 生成密码复制操作。
- 列表/详情中更突出的一键复制操作。
- 更好的备份导入/导出流程，包含文件选择器/共享表和验证结果。
- 收藏夹、最近使用的项目、排序和分组。

## 优先级建议

不要在下一个迭代中交换加密算法。核心加密设计在这个阶段已经很强了。

下一个迭代应该将安全能力转化为用户可见的日常使用产品价值：

1. 添加本地密码健康检查。
2. 在详情、编辑、生成器、剪贴板和锁定转换中强化敏感明文生命周期。
3. 通过预设、密码短语、强度解释和更安全的生成密钥处理来产品化密码生成器。
