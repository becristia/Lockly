# 标签筛选与管理 — 设计说明

**日期：** 2026-05-21

## 概述

为密码库列表添加标签筛选功能。数据模型已有 `tags` 字段，无需修改数据层。新增标签栏 UI + 设置页标签管理。

## 底部标签栏

### 位置

密码库列表页，搜索框下方，搜索结果显示区域上方。

### 行为

- 自动从所有已有条目提取标签，去重排序
- 横向滚动 chips，点击切换选中
- 选中标签后列表仅显示包含该标签的条目
- 点击已选标签取消筛选，显示全部
- 无标签条目隐藏标签栏
- 搜索框文字搜索 + 标签筛选可同时生效（AND 逻辑）

### 视觉

- 水平 `Wrap` 或 `ListView` 排列
- 未选中：`OutlinedChip` 蓝色边框
- 选中：`FilterChip` 蓝色填充
- 左侧可选择加 "全部" chip（默认选中）

## 设置页标签管理

### 入口

设置页新增"标签管理"项，导航到标签管理页。

### 功能

- 列出所有标签（从已有条目提取 + 手动添加的）
- 重命名标签：修改名称后自动更新所有使用该标签的条目
- 删除标签：从所有条目中移除此标签
- 手动添加新标签（预设标签，尚未被条目使用）

### 数据流

标签数据不单独存储。标签列表从已有条目的 `tags` 字段实时计算。手动添加的预设标签可暂时为 0 条目的标签。
